# Environment presets

## configure aws cli

```shell
aws configure
#'us-east-1'
```

# Release

In the project directory, you can run:

`npm run build`

It will generate a release archives `./dist`

`npm run release`

It will upload `./dist` to `s3://website-video-nick/` bucket

# Local development

`npm install`

`npm start`

http://localhost:8000/#/
